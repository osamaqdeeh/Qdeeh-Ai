========================================
   üìù STEP 1: DEPLOY FIRST (Without Env Vars)
========================================

1. In Vercel, click "Deploy" 
   - Framework Preset: Next.js
   - Don't add environment variables yet
   - Just click Deploy

2. The build will complete (might show runtime errors)

3. You'll get a URL like: https://qdeeh-ai.vercel.app

========================================
   ‚öôÔ∏è STEP 2: ADD ENVIRONMENT VARIABLES
========================================

After first deployment:

1. Go to your Vercel Dashboard
2. Click on your project
3. Go to: Settings ‚Üí Environment Variables
4. Add each variable ONE BY ONE:

---

Variable Name: DATABASE_URL
Value: (your database connection string)
Example: postgresql://user:password@host:5432/database
Environment: Production, Preview, Development (select all 3)

---

Variable Name: NEXTAUTH_SECRET
Value: (generate a random secret)
Example: openssl rand -base64 32
Or use: https://generate-secret.vercel.app/32
Environment: Production, Preview, Development (select all 3)

---

Variable Name: NEXTAUTH_URL
Value: https://your-project.vercel.app
(Use the URL Vercel gave you)
Environment: Production

---

Variable Name: GOOGLE_CLIENT_ID
Value: (from Google Cloud Console)
Environment: Production, Preview, Development (select all 3)

---

Variable Name: GOOGLE_CLIENT_SECRET
Value: (from Google Cloud Console)
Environment: Production, Preview, Development (select all 3)

---

Variable Name: STRIPE_SECRET_KEY
Value: sk_test_... or sk_live_...
Environment: Production, Preview, Development (select all 3)

---

Variable Name: STRIPE_PUBLISHABLE_KEY
Value: pk_test_... or pk_live_...
Environment: Production, Preview, Development (select all 3)

---

Variable Name: STRIPE_WEBHOOK_SECRET
Value: whsec_...
(Get from Stripe Dashboard ‚Üí Webhooks)
Environment: Production, Preview, Development (select all 3)

---

Variable Name: NEXT_PUBLIC_APP_URL
Value: https://your-project.vercel.app
Environment: Production

---

Variable Name: MUX_TOKEN_ID
Value: (if using Mux for video)
Environment: Production, Preview, Development (optional)

---

Variable Name: MUX_TOKEN_SECRET
Value: (if using Mux for video)
Environment: Production, Preview, Development (optional)

---

========================================
   üîÑ STEP 3: REDEPLOY
